# Senkou Span (Ichimoku component)

> [!info] Résumé
> Le Senkou Span A et le Senkou Span B sont les deux lignes qui forment le "nuage" (Kumo) de l'Ichimoku Cloud. Elles sont projetées 26 périodes en avant, créant des niveaux de support/résistance futurs qui aident à anticiper les retournements.

## Définition

Le Senkou Span (signifiant "ligne de projection" en japonais) se compose de deux lignes : Senkou Span A et Senkou Span B. Ensemble, elles forment le Kumo (nuage), l'élément le plus distinctif de l'Ichimoku Cloud.

Senkou Span A = (Tenkan-sen + Kijun-sen) / 2, projeté 26 périodes en avant. Senkou Span B = (Plus haut des 52 périodes + Plus bas des 52 périodes) / 2, projeté 26 périodes en avant.

Le "nuage" est l'espace entre le Senkou Span A et le Senkou Span B. Le prix au-dessus du nuage signale une tendance haussière. Le prix en dessous du nuage signale une tendance baissière. Le prix dans le nuage signale une consolidation.

Le Senkou Span B est la ligne la plus lente du système Ichimoku car elle utilise le range des 52 périodes (environ deux mois sur les marchés originaux).

## Contexte et origine

Le Senkou Span est une composante centrale du système Ichimoku Cloud développé par Goichi Hosoda dans les années 1960. La projection 26 périodes en avant était innovante pour l'époque : la plupart des indicateurs sont "rétardaires" car ils analysent le passé, mais le Senkou Span donne une projection future.

Le nuage (Kumo) était l'objectif de Hosoda : créer une zone de support/résistance qui évolue avec le temps. Le nuage change de forme et de position, providing des niveaux de prix dynamiques pour le futur.

En trading moderne, le Senkou Span est particulièrement utile pour identifier les zones de soutien/résistance futures. Le prix qui "projette" vers le nuage peut anticiper une réaction à ce niveau.

## Mécanismes et caractéristiques

Le nuage comme support/résistance : si le prix est au-dessus du nuage, le nuage agit comme support. Si le prix est en dessous du nuage, le nuage agit comme résistance. Plus le nuage est épais, plus le support/résistance est fort.

Le Senkou Span A est plus réactif que le Senkou Span B car il utilise les lignes Tenkan et Kijun (courtes). Le Senkou Span B est très lent car il utilise le range des 52 périodes.

Le croisement du Senkou Span A au-dessus du Senkou Span B forme le "Golden Cross" du nuage. Ce croisement signale un changement de tendance à moyen-long terme. Le croisement inverse (Span A sous Span B) signale un retournement baissier.

Le comportement du prix avec le nuage : le prix qui entre dans le nuage (depuis le haut ou le bas) indique une période de consolidation ou d'incertitude. Le prix qui sort du nuage par le haut est haussier, par le bas est baissier.

## Nuances, critiques, limites

Le Senkou Span est projeté 26 périodes en avant, ce qui le rend moins précis pour les retournements immediats. Il est mieux utilisé pour les niveaux de prix à moyen terme (semaines à mois).

Le Senkou Span B (52 périodes) est très lent à réagir aux changements de prix. Il peut rester stable pendant longtemps, rendant le nuage "épais" même si le marché a changé de comportement.

Le paramètrage de 52 périodes (environ deux mois) peut ne pas être optimal pour le crypto. Des ajustements sont souvent nécessaires selon le timeframe utilisé.

En marché très volatile, le nuage peut être traversé rapidement, réduisant son utilité comme niveau de support/résistance. Le prix peut "exploser" à travers un nuage fin sans réaction.

## Liens et implications

Le [[Ichimoku Cloud]] dans son ensemble intègre le Senkou Span A et B avec le [[Tenkan-sen]], le [[Kijun-sen]], et le [[Chikou Span]]. Chaque ligne a une fonction et elles doivent être lues ensemble.

Les support et résistance formés par le Senkou Span sont uniques car ils sont projetés dans le futur.

Le Senkou Span A et B ensemble forment le "Kumo" (nuage). L'épaisseur du nuage indique la stabilité de la tendance. Un nuage fin signale une tendance faible, un nuage épais signale une tendance forte.
